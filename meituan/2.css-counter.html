1. 在计数前，我们首先要声明一个用来计数的“变量”：counter-reset: number;
  2. 在调用计数前，我们要对这个“计数变量”进行增减：counter-increment: number;
  3. 最后通过 content 来使这个值显示出来：content: counter(number);
  4. 另外，如果将 counter-reset: section; 可以配合subsection进行多级标题的配置
  5. 对于 counter-increment 我们可以让number跳着增加，两种方式：counter-increment: number 2; 或者 counter-increment: number number;
